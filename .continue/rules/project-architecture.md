# Context:
This is a Python package called energy_system_control. It is used to model components of decentralized energy systems, including (but not limiting to) photovoltaic panels, inverters, boilers, heat pumps. The use of this package is to create systems by connecting components together and simulating their behavior over time. The scope is to assess the quality of system designs and, especially, system controls. 

# Architecture
* The package exists as a Github repository at https://github.com/francescobaldi86/EnergySystemControl
* It uses a src layout.
* The main setting file is a pyprojec.toml file
* The testing framework adopted is pytest.
